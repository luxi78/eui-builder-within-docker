EUI Building System On Docker 
====================================================

利用docker构建eui各项手机工程，隔离个版本的环境依赖

Requirement
----------

1. 安装`Docker`并且配置为允许运行`非root`账号运行 
2. 将工程下的bin目录加入`PATH`环境变量中 
3. 当前用户的ssh公钥添加到diana对应账号的ssh key中
